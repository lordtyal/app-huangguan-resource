# app-huangguan-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面。这些页面不针对任何特定域名或网站，仅作为资源归档与分发用途。仓库内容以静态页面为主，适合直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录说明

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各类独立 HTML 页面
├── assets/             # 页面所需的静态资源（CSS、JS、图片等）
└── README.md           # 本文件
```

- `pages/`：按功能或主题分类存放 HTML 文件，每个文件均为独立页面。
- `assets/`：包含页面引用的样式、脚本及媒体文件，建议按子目录整理。

## 页面归档说明

- 所有页面均为独立 HTML，不依赖后端服务。
- 页面之间的链接或跳转仅用于本地导航，不指向外部特定资源。
- 归档内容会不定期更新，新增或调整页面时会在提交信息中注明。

## 维护说明

- 本仓库仅供归档与展示，不接受功能请求或定制开发。
- 如发现页面无法正常显示，请检查资源路径是否正确，或确认浏览器支持相关特性。
- 如需引用本仓库内容，请保留原始出处信息，并遵守开源协议（如有）。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可浏览。
3. 若部署到 GitHub Pages，请在仓库设置中启用 Pages 服务，选择主分支作为源。
